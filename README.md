# AI-Powered-ESG-Report-Analysis-System
-	Built a RAG-based analysis pipeline for Tesla's 2024 ESG Impact Report using LangChain, Pinecone, and HuggingFace
-	Chunked and embedded 187 pages of PDF data using sentence transformers; stored vectors in Pinecone for semantic search
-	Generated topic summaries across 8 ESG categories using Flan-T5-Large; visualized embedding clusters in interactive 3D PCA plots
